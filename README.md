Villa Holdings — HR Management Portal
Project Overview
A fully responsive Human Resources Management Portal website built for Villa Holdings, a technology company based in Maseru, Lesotho. This project was developed as part of a web design assignment at Limkokwing University of Creative Technology.

Live Website
🌐 View Live Site

Pages
Home (index.html) — Main landing page with company overview
Company Culture (culture.html) — Team members, values and work environment
Recruitment & Careers (recruitment.html) — Job listings and hiring process
Employee Benefits (benefits.html) — Full benefits package information
Training & Development (training.html) — Training programs and embedded HR video
HR Policies (policies.html) — Internal company policies with anchor links
Employee Login (login.html) — Staff login portal with logout functionality
Features
✅ Responsive design — works on mobile and desktop
✅ Fixed navigation menu on all pages
✅ Job vacancy listings
✅ Embedded HR training video
✅ Staff images
✅ Internal links to HR policies
✅ Employee login and logout
✅ Smooth hover animations
